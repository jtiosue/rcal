Installation
------------

To install:

.. code:: shell

    pip install git+https://github.com/jtiosue/rcal

Or you can clone the repository and then install:

.. code:: shell

    git clone https://github.com/jtiosue/rcal.git
    cd rcal
    pip install -e .
